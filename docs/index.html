zohopay TypeScript SDK

Docs

Large-scale, extensible Zoho Payments SDK focused on authentication, payment tokens, and payment verification.

What is included

  • OAuth URL generation
  • Authorization-code token exchange
  • Refresh-token access token generation
  • Payment links
  • Payment sessions / tokens
  • Payment verification
  • Customers and refunds
  • Pluggable HTTP transport

Files

  • src/auth — OAuth helpers and token lifecycle
  • src/net — transport and authenticated client
  • src/services — API resources
  • src/models — response models
  • src/params — request serializers + validation
  • examples/playground.html — browser test page

Install

bun install
bun run build

Create a client

import { ZohoPayments, Edition } from "./dist/index.mjs";

const client = ZohoPayments.builder()
  .accountId("23137556")
  .edition(Edition.IN)
  .accessToken("1000.xxxx.yyyy")
  .build();

Authentication flow

1. Build auth URL

import { buildAuthorizationUrl, Edition, OAUTH_SCOPES } from "./dist/index.mjs";

const url = buildAuthorizationUrl({
  clientId: "1000.YOUR_CLIENT_ID",
  accountId: "23137556",
  scopes: [
    OAUTH_SCOPES.PAYMENTS_CREATE,
    OAUTH_SCOPES.PAYMENTS_READ,
    OAUTH_SCOPES.REFUNDS_CREATE,
    OAUTH_SCOPES.REFUNDS_READ,
  ],
  redirectUri: "https://yourapp.example.com/oauth/callback",
  edition: Edition.IN,
  accessType: "offline",
});

2. Refresh access token

const fresh = await ZohoPayments.generateAccessToken({
  refreshToken: "1000.refresh_token",
  clientId: "1000.YOUR_CLIENT_ID",
  clientSecret: "your_client_secret",
  redirectUri: "https://yourapp.example.com/oauth/callback",
  edition: Edition.IN,
});

client.updateOAuthToken(fresh);

3. Parse callback and exchange auth code

const callback = ZohoPayments.parseOAuthCallback(
  "https://yourapp.example.com/oauth/callback?code=1000.xxx&state=my-state"
);

if (!callback.code) {
  throw new Error("Missing authorization code");
}

const tokenSet = await ZohoPayments.exchangeCodeForToken({
  code: callback.code,
  clientId: "1000.YOUR_CLIENT_ID",
  clientSecret: "your_client_secret",
  redirectUri: "https://yourapp.example.com/oauth/callback",
  edition: Edition.IN,
});

console.log(tokenSet.accessToken, tokenSet.refreshToken);

4. Revoke refresh token

await ZohoPayments.revokeToken({
  token: "1000.refresh_token",
  edition: Edition.IN,
});

Payment token / session

const session = await client.paymentSessions().create({
  amount: 250,
  currency: "INR",
  description: "Checkout session",
  expiresIn: 600,
});

console.log(session.paymentsSessionId, session.accessKey);

Payment verification

const payment = await client.payments().verify("payment_id_here");
console.log(payment.status, payment.amount, payment.currency);

Payment link

const link = await client.paymentLinks().create({
  amount: 500,
  currency: "INR",
  description: "Order #1234",
  email: "customer@example.com",
});

console.log(link.paymentLinkId, link.url);
